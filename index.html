Real-Time News Sentiment Analysis System

Led the development of an innovative real-time news sentiment analysis system, combining traditional machine learning with modern data streaming architecture. The project achieved 92% accuracy in sentiment classification using Logistic Regression, while implementing Kafka for real-time data processing. Key Achievements: Developed an end-to-end pipeline from web scraping to real-time sentiment prediction Implemented Snorkel technique for efficient data labeling Built a scalable system using Apache Kafka for real-time news processing Created an interactive UI for sentiment visualization Tech Stack: Python, Apache Kafka, Machine Learning (Logistic Regression), NLP, Web Scraping, TextBlob, Snorkel This project demonstrates practical application of big data analytics in news sentiment analysis, offering valuable insights for media monitoring and market analysis.

Product Extension Analyzer

Created an advanced Python routine utilizing BeautifulSoup and Selenium to automate web scraping, aimed at extracting detailed data on products and their usage from multiple websites. This routine employs sophisticated data extraction techniques to ensure efficient and accurate data retrieval. It includes features to analyze and ascertain crop seasonality associated with these products, thereby providing valuable insights for agricultural decision-making. The routine is designed to handle multilingual websites, leveraging Google Translate for accurate translations, ensuring relevant data is consistently identified and extracted. It integrates a user-friendly subroutine, allowing seamless and scalable data extraction across various country-specific websites. The project underscores the importance of automation in managing large-scale web scraping tasks, showcasing significant improvements in data collection efficiency and reliability..

Brown Munde Car Rental System

Our team has successfully developed a robust Car Rental System as part of our Object-Oriented Programming Lab Mini Project, utilizing Java along with Swings and AWT for an enhanced user interface. Employing comprehensive Object-Oriented Programming (OOP) concepts such as Inheritance, Packages, Exception Handling, Generics etc , we have crafted a beta version of the car reservation system that not only fulfills its immediate purpose but also lays the foundation for future enhancements and expansions.
